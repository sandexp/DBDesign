#### 删除表

删除表的相关资源和数据：

##### 总体思路
一个表具有相关的表空间, 需要删除表空间相关的描述信息, 例如表的schema信息. 同时需要从元数据表中删除表相关的内容.

在此基础之上，需要删除表的数据. 表空间内包含有若干个数据页, 在这里使用标记删除的方式, 将数据表中的数据页中载入内存标记删除.


1. 元数据中关于数据表的移除
在db.h中维护了一个打开表的列表， 将删除的表删除即可

数据结构为
```c++
std::unordered_map<std::string, Table *>  opened_tables_
```

但是没有提供相应的相应的api 需要自己实现

2. 关于表中数据的删除

表中数据保护三个部分
+ 数据部分
每个表的数据会生成tableName.data的数据文件，需要从磁盘中删除掉

+ 表空间描述符
表的描述符新为tableName.table 需要从磁盘中移除

+ 索引
如果表中建立有索引, 名称为tableName.index 删除掉


3. 清理缓冲池中表相关数据
每个数据表中都维护了一个缓冲池实例`DiskBufferPool`,这时候只需要调用对应表的析构函数, 释放表中资源即可
